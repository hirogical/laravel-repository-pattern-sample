## 概要
Laravel×リポジトリパターンのサンプルです。

リレーションの学習にも使えるよう、

- 一対多(hasMany)
- 多対一(belongsTo)
- 多対多(belongsToMany)

を網羅しています。

## 使用技術
- PHP 8.0.15
- Laravel 8.81.0
